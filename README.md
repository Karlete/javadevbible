# ☕ JavaDev Bible

**Your Comprehensive Java Developer Knowledge Base**

A complete, organized reference covering everything from Java fundamentals to enterprise development — all in one place, accessible offline.

## 🔗 Live Demo

[Live Demo](https://Karlete.github.io/javadevbible)

## ✨ Key Features

- **90+ Topics** covering the full Java ecosystem
- **Instant Search** across all topics, concepts, and keywords
- **Categorized Sections** — Fundamentals, Advanced, Spring, Jakarta EE, Databases, Security, Build Tools, and more
- **Syntax-Highlighted Code Examples** in every topic
- **Beautiful Animations** — animated gradient hero, floating code particles, typing effect, smooth card hover effects
- **Fully Static** — no backend, no build step, just HTML/CSS/JS with vanilla JavaScript
- **Mobile Responsive** — works on any device with responsive canvas and adaptive layouts
- **Offline Ready** — download and use without internet

## 🚀 How to Use Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Karlete/javadevbible.git
   ```
2. **Open the site:**
   ```
   Open javabible/index.html in your browser
   ```
   No server, build step, or dependencies required.

## 📦 Download ZIP

1. Click the green **Code** button on the GitHub repository page
2. Select **Download ZIP**
3. Extract the ZIP file
4. Open `javabible/index.html` in your browser
