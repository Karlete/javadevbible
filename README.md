# ☕ JavaDev Bible

**Your Comprehensive Java Developer Knowledge Base**

A complete, organized reference covering everything from Java fundamentals to enterprise development — all in one place, accessible offline, with zero backend and zero build step.

## 🔗 Live Demo

[Live Demo](https://karlete.github.io/javadevbible/javabible/index.html)

## ✨ Key Features

- **90+ Topics** covering the full Java ecosystem
- **Instant Search** across all topics, concepts, and keywords
- **Categorized Sections** — Fundamentals, Advanced, Spring, Jakarta EE, Databases, Security, Build Tools, and more
- **In-page Table of Contents** on every topic, with collapsible navigation
- **Junior + Senior Interview Questions** on every topic — dual-level content for developers at any stage
- **Syntax-Highlighted Code Examples**, all using a consistent e-commerce domain so examples read like a real, coherent codebase rather than disconnected snippets
- **i18n-Ready** — every rewritten page is scaffolded for translation (see Roadmap)
- **Beautiful Animations** — animated gradient hero, floating code particles, typing effect, smooth card hover effects
- **Fully Static** — no backend, no build step, just HTML/CSS/JS with vanilla JavaScript
- **Mobile Responsive** — works on any device with responsive canvas and adaptive layouts
- **Offline Ready** — download and use without internet

## 📊 Project Status

This project is under active, topic-by-topic quality improvement — older topics are being rewritten to a consistent standard: production-grade depth, a single running e-commerce domain across all code examples, in-page TOC, and junior/senior interview questions.

**Fully modernized categories:**
- ✅ Tools & Ecosystem (IDEs, Git, CI/CD, Logging, Testing, Code Quality Tools)
- ✅ Best Practices (SOLID, Clean Code, Error Handling, Performance, Security, Code Conventions, SonarQube & Code Quality)

**In progress:** Databases & Persistence, Jakarta EE, Application Servers & Deployment

**Not yet modernized:** Fundamentals, Advanced Java, Spring Framework, Build Tools, Web Concepts, Java Versions

If you land on a page from an unmodernized category, expect solid reference content, but not yet the same depth as the categories above — that gap is closing incrementally.

## 🚀 How to Use Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Karlete/javadevbible.git
   ```
2. **Open the site:**
   ```
   Open javabible/index.html in your browser
   ```
   No server, build step, or dependencies required.

## 📦 Download ZIP

1. Click the green **Code** button on the GitHub repository page
2. Select **Download ZIP**
3. Extract the ZIP file
4. Open `javabible/index.html` in your browser
